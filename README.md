## testcase 

目录包含了Lab1的测试用例

测试用例从简单到困难进行排列

#### 手动测试方法

将某一个测试用例的imem.txt和dmem.txt拷贝到目标目录下并运行程序，程序完成后将生成的dmemresult.txt，RFresult.txt和stateresult.txt与测试用例下的同名文件进行对比

## assembler

目前支持的指令有ADDU, SUBU, LW, SW, BEQ, HALT
